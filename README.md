# SupportBotTG

SupportBot is a Telegram-based appointment booking and notification system designed to improve the workflow for support and consultation teams. The bot allows users to schedule appointments directly through Telegram, receive reminders 24 hours before the meeting, and confirm their attendance, which helps staff manage time and availability more efficiently.

## Features

- Schedule an appointment through the Telegram bot
- Automated 24-hour reminder system via scheduler
- Confirmation of attendance via Telegram
- REST API for managing appointments and users
- Full CRUD functionality: Create, Read, Update, Delete
- Integration with PostgreSQL via Docker
- Automated database migration with Liquibase
- Swagger (OpenAPI) documentation
- Unit testing with JUnit

## Technologies Used

- Java 19+
- Spring Boot
- Spring Data JPA (Hibernate)
- Spring Scheduler
- Lombok
- Liquibase
- OpenAPI (Swagger UI)
- PostgreSQL (via Docker)
- JUnit 5
- Maven
